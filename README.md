# Encriptador de Textos

Este proyecto es una aplicación web que permite encriptar y desencriptar textos usando un conjunto de reglas específicas. La aplicación está construida con HTML, CSS y JavaScript.

## Características

- Encriptación de texto basada en reglas específicas.
- Desencriptación de texto encriptado a su forma original.
- Funciona solo con letras minúsculas y sin caracteres especiales.
- Interfaz amigable con campos de texto para entrada y salida.
- Botones para encriptar, desencriptar y copiar texto.
- Limpieza automática del campo de resultado después de copiar el texto.

## Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari).

## Instalación

1. Clona el repositorio o descarga los archivos del proyecto.
    ```bash
    git clone https://github.com/tu-usuario/encriptador-de-textos.git
    ```
2. Navega a la carpeta del proyecto.
    ```bash
    cd encriptador-de-textos
    ```

## Uso

1. Abre el archivo `index.html` en tu navegador web.
2. Introduce el texto que deseas encriptar o desencriptar en el campo de entrada.
3. Haz clic en el botón correspondiente para encriptar o desencriptar el texto.
4. El resultado se mostrará en el campo de salida.
5. Puedes copiar el resultado haciendo clic en el botón "Copiar".
